How to use:

  • Select bingo size (from 2x2 to 8x8 grid).
  • Open each category to select prompts of your choice.
  • Warning: angst/whump prompts mention psychological and physical violence.
  • Writing prompts contain writing rules, most suited for fanfictions.
  • Visual prompts are more suited for fan art, in any medium.
  • Selected categories are evenly represented.
  • If the grid isn't completely filled, make sure to have selected enough prompts.

Usage suggestions:

  • On a 2x2 grid, select Character, angst, fluff, or themed prompts and either visual or writing. Create a work that fits all 4 elements.
  • On a mid-sized grid (3x3, 4x4 or 5x5), select a few different categories. Create works that fulfill as much of the grid as possible and mark the fulfilled prompts. Keep going until the grid is fully marked!
  • On a bigger grid (6x6 to 8x8), draw vertical, horizontal or diagonal lines and fulfill all the prompt crossed that way by a single fanwork.
  • On a mid-sized or big grid, select only characters and one type of prompts. Create a work that feature each pair of character found next to each other alongside the nearest prompt.
  • Anniversary Special! Pick a 6x6 grid with a mix of FFIV elements and prompts. Make pairs across the grid, create a small work for each pair, and post them from July 1st to July 19th (FFIV's anniversary).
